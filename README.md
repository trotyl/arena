## 需求

### 第一问

* 对打游戏
* 有两个人，下称玩家
* 每个人都有血值
* 有攻击力
* 没有防御力
* 有名字
* 可以互相对打
* 游戏开始，双方互殴，你一下我一下，直到一人死亡，打印出xx被打败了
* 只需要写核心逻辑，不需要考虑界面

输出

```
李四被打败了
```

### 第二问

* 每次攻击输出谁攻击了谁，被攻击的人掉了多少血，剩多少血。

输出:

```
张三攻击了李四,李四受到了8点伤害,李四剩余生命：12
李四攻击了张三,张三受到了9点伤害,张三剩余生命：1
张三攻击了李四,李四受到了8点伤害,李四剩余生命：4
李四攻击了张三,张三受到了9点伤害,张三剩余生命：-8
张三被打败了
```
